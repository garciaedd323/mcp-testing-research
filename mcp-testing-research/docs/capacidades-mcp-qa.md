# Capacidades de Claude + MCP en QA

## 🟢 Exploración y creación

### Exploración web automática
Navega, inspecciona y mapea selectores automáticamente en una web, sin necesidad de escribirlos a mano.

### Generación de suites
Genera suites de tests completas a partir de una descripción en lenguaje natural.

### Auto-mantenimiento
Detecta tests rotos (por ejemplo, por cambios en el DOM) y los repara automáticamente.

## 🔵 Análisis e integración

### Análisis de reportes
Identifica patrones de fallo y tendencias a partir de reportes de ejecución de tests.

### Integración CI/CD
Analiza resultados de tests directamente dentro del pipeline de integración continua.

## 🟠 Ejecución y cobertura

### Generación de datos
Crea datos de prueba masivos y realistas para poblar tests.

### Auditoría de accesibilidad
Ejecuta auditorías a11y automáticas usando Playwright.

### Testing cross-browser
Corre los mismos tests en Chrome, Firefox y Safari en paralelo.
